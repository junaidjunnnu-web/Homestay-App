Build a homestay booking platform called "Homestay App" for Indian homestays 
(especially places like Coorg, Ooty, Munnar, Goa)etc.

TWO USER ROLES:
1. HOST (homestay owner) - manages their property, rooms, bookings
2. GUEST (traveler) - browses ALL properties, books rooms

CRITICAL RULE: Guest browse screen must show ALL properties from ALL hosts 
with NO user-based filtering. Only the host dashboard filters by hostId.

DATABASE:
- Users: id, name, email, mobile, role (host/guest), createdAt
- Properties: id, hostId, name, address, city, state, photos, description, 
  amenities, mealsIncluded, locationLat, locationLng, nearbyAttractions, 
  upiId, bookingMode (inquiry/instant), cancellationPolicy, status (active/inactive)
- Rooms: id, propertyId, name, type, pricePerNight, capacity, photos, status
- Bookings: id, roomId, guestId, checkIn, checkOut, status 
  (pending/confirmed/cancelled/completed), totalAmount, paymentStatus, 
  specialRequests, referenceNumber, createdAt
- Reviews: id, propertyId, guestId, rating, comment, createdAt
- Payments: id, bookingId, amount, method, status, transactionId

HOST FEATURES:
1. Register/login with mobile OTP or email
2. Add property with photos, location, amenities, meals info, nearby attractions
3. Add rooms with photos, pricing, capacity
4. Dashboard: occupancy %, revenue this month, today check-ins/check-outs
5. Approve or decline pending booking requests
6. Calendar view of all bookings
7. WhatsApp notification when new booking arrives (use Linking to open 
   WhatsApp with pre-filled message)
8. Guest management with contact details
9. Payment tracking (paid vs pending)
10. Housekeeping and maintenance task tracking
11. Toggle per property: Inquiry Mode vs Instant Mode

GUEST FEATURES (no login required to browse):
1. Home screen: search by city, date range, guest count
2. Browse ALL active properties - show property photo, name, city, price/night, rating
3. Property detail page: photos gallery, amenities, meals included, 
   nearby attractions, location map pin with "Get Directions" button, 
   room list with prices, guest reviews
4. Real-time availability calendar - already booked dates greyed out
5. Booking flow - TWO MODES based on host setting:

   INQUIRY MODE:
   - Guest fills name, mobile, email, dates, guest count
   - Submit request → status = pending
   - Guest gets reference number to track status
   - Host gets WhatsApp notification to approve/decline

   INSTANT MODE:
   - Guest picks dates, sees price breakdown
   - Pays via Razorpay (UPI, cards, netbanking) 
   - Booking auto-confirmed immediately
   - Host notified via WhatsApp (for awareness, no approval needed)

6. Check booking status by reference number (no login needed)
7. Leave review after checkout
8. Guest can book without creating account (auto-create account in background)

UI REQUIREMENTS:
- Mobile-first React Native (Expo) app, also works as website
- Warm, trustworthy color scheme (greens/earthy tones like #1B6B5A)
- Professional clean UI similar to OYO/Airbnb style
- India-specific: INR pricing, UPI payment, Indian city names in search

TECH:
- Expo Router (file-based routing)
